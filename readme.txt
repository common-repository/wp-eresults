== Description ==

This plugin adds a widget in the sidebar that shows the results of the Euromillions Lottery from http://www.en15.com website. Each time a visitor to your Blog Euromillions plays (Euromillions or another lottery games), you win up to 3% (sometimes more) of what the visitor has played and will continue to gain each time you play (all games, not euromillions only), and visitors without the need to revisit your blog.
	
Available in English and Spanish. Internationalize by using files. mo

Compatible with Wordpress 2.7 version (tested in 2.6, 2.6.1, 2.7, 2.7.1)

== Installation ==

= English: =

Installing the plugin requires WordPress Theme used in support widgets.

Simply download the Zip file and extract the folder containing the files of the plugin in the WordPress directory "wp-content/plugins/".

You should see "wp-content/plugins/wp-eresults"

Join the WordPress administration page, click on Plugins and activate it.

After this you should click Appearance> Widgets and add the widget to the toolbar of your choice.

From here you can also set some parameters for appearance of the widget.

Readme  > http://english.wpresults.com/install


= Spanish: =

La instalacion del plugin requiere que el Tema utilizado en WordPress soporte widgets.

Simplemente descargue el archivo Zip y extraiga la carpeta que contiene los archivos del plugin en el directorio de WordPress "wp-content/plugins/".

Ahora deberias ver "wp-content/plugins/wp-eresults"

Entre en la pagina de administracion de WordPress, haga clic en Plugins y activelo.

Despues de esto debera pulsar en Apariencia>Widgets y añadir el widget a la barra de herramientas de su preferencia.

Desde aqui tambien podra configurar algunos parametros de apariencia del widget.

Leer  > http://wpresults.com/instalacion
